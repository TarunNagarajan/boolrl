# Changelog

## [1.0.0] - 2025-11-05

### Added
- Initial release of Boolean Expression Simplification using Deep Reinforcement Learning
- Three different agent architectures: MLP-based, Sequence-based (LSTM), and GNN-based
- Support for training and testing all three agent types
- Comprehensive documentation and configuration files

### Features
- Boolean expression generation and simplification using SymPy
- DQN implementation with experience replay and target networks
- Support for multiple reward function designs
- Training visualization and checkpointing capabilities

### Bug Fixes
- Fixed critical batch processing bug in GNN agent
- Fixed variable unpacking in GNN environment
- Fixed undefined constants in test files
- Fixed import statement issues

### Documentation
- Reward function design evolution documentation
- Training summary and future directions
- Comprehensive README with installation and usage instructions
